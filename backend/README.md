# Backend

basic firebase backend in python

## Requirements

- [uv](https://github.com/astral-sh/uv) (recommended package + venv manager)
- python 3.13

## Setup

```bash
# sync dependencies
cd backend
uv sync

# activate environment
source .venv/bin/activate   # macos/linux
.venv\Scripts\activate      # windows

# run app
uv run main.py
```
